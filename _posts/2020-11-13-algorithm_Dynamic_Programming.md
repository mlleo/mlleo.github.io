#### 피보나치 수열 

`code`
```py
# 피보나치 함수를 재귀함수로 구현
def fibo(x):
  if x == 1 or x == 2:
    return 1
  return fibo(x-1) + fibo(x-2)
```
- 시간 복잡도 : O(2^N)
