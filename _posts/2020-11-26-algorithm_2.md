### 그리디 알고리즘

#### 문제1


`code`
```py
n = int(input())
stat = list(map(int, input.split()))
stat.sort()

group = 0

count = 0

for i in stat:
	count += 1
	if count >= i:
		group += 1
		count = 0

print(group)
```

data = input()
result = int(data[0])

for i in range(1,len(data)):
	num = int(data[i])
	if num<= 1 or result <=1:
